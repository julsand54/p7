# p7

CV responsive para viewport de ordenador y móvil.

## Descripción

Página web interactiva que utiliza HTML y CSS para crear un diseño visual moderno con patrones geométricos. El proyecto implementa un sistema de contenedores con elementos decorativos ("pétalos") usando colores vibrantes.

## Información del Proyecto

| Propiedad | Valor |
|-----------|-------|
| **Creator** | Julia |
| **Year** | 2025 |
| **Language** | HTML, CSS |
| **Type** | Responsive Web Design |

## Características

- **Diseño Responsive**: Compatible con viewport de ordenador y dispositivos móviles
- **Paleta de Colores**: 
  - Amarillo (#FFCF4A)
  - Naranja (#FE763C)
  - Azul Profundo (#281F77)
  - Gris (rgb(104, 104, 104))
- **Tipografía**: Adobe Typekit integrado
- **Estructura Modular**: Contenedores dinámicos con elementos decorativos
- **CSS Avanzado**: Variables CSS, posicionamiento absoluto y flexibilidad de viewport

## Tecnologías

- HTML5
- CSS3 (Variables CSS, Grid Layout)
- Adobe Typekit (Fuentes Web)
